# ⚔️ Rucoy Online: Guild Ranking & Evolution Tracker

Este projeto nasceu da necessidade de monitorar de forma mais eficiente a progressão de personagens e o desempenho coletivo de guildas no jogo **Rucoy Online**. Atualmente, o projeto está em sua fase inicial de desenvolvimento (Front-end).

> [!NOTE]
> **Status do Projeto:** 🏗️ Em desenvolvimento (Estrutura HTML finalizada).

---

## 🎯 Objetivo do Projeto

O site visa substituir métodos manuais de acompanhamento de status por uma interface web intuitiva onde os jogadores podem:
* Visualizar o ganho de XP (Level e Skills) em intervalos mensais.
* Participar de um sistema de ranking interno para guildas.
* Registrar a evolução pessoal de forma simples e visual.

---

## ✨ Funcionalidades Planejadas

- [x] **Estrutura Base:** HTML5 semântico para organização dos dados.
- [ ] **Entrada de Dados Inteligente:** Substituição de campos de texto por **Sliders (intervalos de 10%)** para reportar progresso de skills, melhorando a UX.
- [ ] **Dashboard de Evolução:** Gráficos ou tabelas que mostram o crescimento de status ao longo do tempo.
- [ ] **Sistema de Ranking:** Algoritmo para classificar membros da guilda com base no ganho de XP mensal.

---

## 🛠️ Tecnologias Utilizadas (Stack)

* **HTML5:** Estruturação de conteúdo.
* **CSS3:** (Planejado) Estilização com foco em uma interface temática do jogo.
* **JavaScript:** (Planejado) Lógica de cálculo de XP, manipulação dos sliders e geração de rankings.

---

## 🚧 Por que não usar OCR?

Durante o planejamento, foi considerada a ideia de utilizar leitura de imagem (OCR) para automatizar a captura de dados através de prints do jogo. No entanto, a abordagem foi descartada por ser considerada pouco eficiente para o escopo atual, optando-se por uma entrada de dados manual mais fluida e validada.
